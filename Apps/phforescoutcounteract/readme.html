This app requires CounterACT 8.0 or above with the Open Integration Module installed. When the Open Integration Module is installed, Web API and Data Exchange (DEX) modules will be available. Web API is a read-only module, while DEX is a read and write module.
In the case of restricting updates to CounterAct, only supply Web API credentials in the asset configuration.

Playbook Backward Compatibility

  • get device info - 2 new action parameters 'host_ip' and 'host_mac' are added. Hence, it is requested to the end-user to please update their existing playbooks by modifying the corresponding action blocks for this action on the earlier versions of the app.
